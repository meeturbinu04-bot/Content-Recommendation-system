# Context-Aware Movie Recommendation System

This project recommends movies based on the user's current mood and intent using Large Language Models (LLMs). 
The system provides explainable recommendations and can be extended for other content types.

## Project Files
- `Context_Aware_Content_Recommendation_System.ipynb` : Main notebook
- `movies.csv` : Movie metadata
- `requirements.txt` : Python dependencies

## How to Run
1. Clone the repo
2. Place `movies.csv` in the same folder as the notebook
3. Open the notebook and run all cells
